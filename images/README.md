# Images pour EnHarmonie

## Images requises

Pour que le site fonctionne parfaitement, vous devez placer les images suivantes dans ce dossier :

### 1. piano-keys.jpg
- **Utilisation** : Image de fond pour la section Hero (en-tête)
- **Dimensions recommandées** : 1920x1080px minimum
- **Description** : Photo de touches de piano en noir et blanc, avec un bel effet de profondeur
- **Source** : Utilisez l'image fournie du clavier de piano

### 2. piano-sheet.jpg
- **Utilisation** : Image dans la section "À propos"
- **Dimensions recommandées** : 800x600px minimum
- **Description** : Photo d'un piano avec une partition musicale dans une ambiance douce et chaleureuse
- **Source** : Utilisez l'image fournie du piano avec partition

## Instructions

1. Renommez vos images selon les noms ci-dessus
2. Placez-les dans le dossier `/images/`
3. Les images seront automatiquement chargées par le site

## Optimisation recommandée

Pour de meilleures performances, optimisez vos images avant de les utiliser :
- Format JPEG avec qualité 80-85%
- Compression avec des outils comme TinyPNG ou ImageOptim
- Dimensions adaptées (pas trop grandes)

## Images alternatives

Si vous n'avez pas accès aux images originales, vous pouvez utiliser :
- Unsplash : https://unsplash.com/s/photos/piano
- Pexels : https://www.pexels.com/search/piano/

Recherchez des images avec :
- Ambiance sombre et élégante
- Éclairage doux et intimiste
- Haute résolution
- Licence libre de droits
